# ESP32+ PROJECT

### Features
- Two Pushbutton with pull up resistor.
- Four 5V LED 5mm.
- 5V Buzzer DC
- Additional OLED Display 128x32
- Power Expander 5V and 3.3V
- External Power & Boot Mode Fixed
